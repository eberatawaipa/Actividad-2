1- ¿Cuáles son las principales funciones de un kernel en la arquitectura de un sistema operativo y cómo interactúa con el hardware?

Funciones del Kernel:

El kernel es como el jefe del sistema operativo. Hace tres cosas importantes: Maneja la memoria: Decide quién usa qué parte de la memoria. Controla los procesos: Decide cuándo y cuánto tiempo cada programa puede usar la computadora. Habla con los dispositivos: Como un intermediario entre el software y el hardware.

3- ¿Qué medidas de seguridad se pueden implementar a nivel de sistema operativo para proteger una red, especialmente en entornos empresariales?

Actualización constante:

Mantén tu sistema operativo y aplicaciones al día. Las actualizaciones suelen incluir parches de seguridad que corrigen vulnerabilidades.

Restricción de acceso:

Limita el acceso a la información sensible. Solo los usuarios autorizados deben tener permisos para acceder a datos críticos.

Copia de seguridad:

Realiza copias de seguridad periódicas de tus datos importantes. Así, en caso de un incidente, podrás restaurar la información sin pérdidas significativas.

Contraseñas seguras:

Utiliza contraseñas fuertes y cámbialas regularmente. Evita contraseñas obvias como “123456” o "contraseña".

Firewalls:

Instala y configura un firewall. Bloquea el tráfico no autorizado y permite solo el tráfico necesario para los servicios conocidos
